  • Event Listeners:-
    • click - when the element clicked
    • dblclick - when the element double clicked
    • mouseenter - when the mouse point enter to the element
    • mouseleave - when the mouse pointer leave the element
    • mousemove - when the mouse pointer move on the element
    • mouseover - when the mouse pointer move on the element
    • mouseout -when the mouse pointer out from the element
    • input -when value enter to input field
    • change -when value change on input field
    • blur -when the element is not focused
    • keydown - when a key is down
    • keyup - when a key is up
    • keypress - when we press any key
    • onload - when the browser has finished loading a page
  • difference bettween mouseover,mouseenter and mousemove
    • Mouseover :-
      • This event occurs every time when the mouse pointer enters the div element or its child elements.
    • Mouseenter :-
      • This event occurs only when the mouse pointer enters the div element.
    • Mousemove :-
      • This event occurs every time the mouse pointer is moved over the div element.
