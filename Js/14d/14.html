• The try statement defines a code block to run.
  • The catch statement defines a code block to handle any error.
  • The finally statement defines a code block to run regardless of the result.
  • The throw statement defines a custom error.
  • Error handling :-
    • when something happen other than the given condition or task and than error comes is if known as error handling.
    • when something happen other than the given task or expected condition's result comes up in try block which will cause error and that error will be shown in catch block.
  • Type Error name:-
    • EvalError = An error has occurred in the eval() function
    • RangeError = A number "out of range" has occurred
    • ReferenceError =An illegal reference has occurred
    • SyntaxError = A syntax error has occurred
    • TypeErrot = A type error has occurred
    • URIError = An error in encodeURI() has occurred
